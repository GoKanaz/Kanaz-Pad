## [1.0.0] - 2024-XX-XX

### Added
- Initial release of Kanaz Pad
- Markdown editor with syntax highlighting
- Real-time preview
- Local storage with Room Database
- Note organization with folders
- Search functionality
- Export capabilities (PDF, HTML, DOCX)
- Import from markdown and text files
- Version history tracking
- Autosave feature
- Focus mode for distraction-free writing
- Image and file attachments
- Material Design 3 UI
- Dark theme support

### Developer Notes
- Built with Kotlin
- MVVM architecture pattern
- Room Database for local storage
- Markwon library for markdown rendering
- iText7 for PDF generation
- Apache POI for DOCX export
- Kotlin Coroutines for async operations

## Release Notes Format

### Version Numbering
- MAJOR version: Incompatible API changes
- MINOR version: Backwards-compatible functionality
- PATCH version: Backwards-compatible bug fixes

### Categories
- **Added**: New features
- **Changed**: Changes in existing functionality
- **Deprecated**: Soon-to-be removed features
- **Removed**: Removed features
- **Fixed**: Bug fixes
- **Security**: Vulnerability fixes

### Example Entry

```markdown
## [1.1.0] - 2024-XX-XX

### Added
- Cloud synchronization with Google Drive
- Collaborative editing support
- Export to Notion format

### Changed
- Improved markdown parser performance
- Updated UI design for better accessibility

### Fixed
- Fixed crash when exporting large notes
- Resolved issue with special characters in file names

### Security
- Added end-to-end encryption for cloud sync
```
